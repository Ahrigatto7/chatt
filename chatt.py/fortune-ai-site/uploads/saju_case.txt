# 자녀

- **사주원국에서 자식궁과 자식성을 찾고, 자식의 성별을 구분하는 방법 총정리**
    
    **1. 자식궁(子息宮)과 자식성(子息星)의 정의**
    
    **•	자식궁(子息宮): 일주의 시지(時支)를 자식궁으로 본다.**
    
    **•	자식궁의 십신(十神)과 지장간(地藏干)에 숨겨진 자식성을 분석하여 자녀운을 판단한다.**
    
    **•	자식성(子息星): 사주의 십신(十神) 중 자식을 나타내는 요소.**
    
    **•	남명(男命) → 정관(正官) 또는 편관(偏官) 이 자식성.**
    
    **•	여명(女命) → 식신(食神)과 상관(傷官) 이 자식성.**
    
    **•	자식성이 투출(透出) 되었는지, 지장간(地藏干)에 숨어 있는지에 따라 자식의 존재 여부와 성별을 판단.**
    
    **2. 자식의 성별 구분법**
    
    **자녀의 성별은 명주의 성별(남명/여명)에 따라 다르게 정해진다.**
    
    **① 남명(男命)의 경우**
    
    **•	정관(正官)과 편관(偏官)이 자식성이며, 정관과 편관이 동시에 있으면 정관을 우선으로 본다.**
    
    **•	자식성(관성)의 음양에 따라 성별 결정:**
    
    **•	양(陽) 관성 → 아들**
    
    **•	음(陰) 관성 → 딸**
    
    **•	만약 관성이 없으면 재성(財星)을 자식으로 본다.**
    
    **•	남명에서 관성이 생한 인성(印星)도 자식으로 본다.**
    
    **•	남명에서 자식성이 자식궁(時支)에 있으면 음양에 관계없이 아들이 된다.**
    
    **•	남명에서 자식성이 재성의 생(生)을 받으면 아들, 생을 받지 못하면 딸이 된다.**
    
    **예시: 甲戌日柱가 戌(지장간: 辛, 丁, 戊)을 포함할 때→辛(陰官)이므로 딸이지만,戊土(財)가 생(生)하면 아들로 변함.**
    
    **② 여명(女命)의 경우**
    
    **•	식신(食神)과 상관(傷官)이 자식성으로, 식신과 상관의 음양에 따라 성별이 결정된다.**
    
    **•	양(陽) 식신 → 아들**
    
    **•	음(陰) 식신 → 딸**
    
    **•	상관(傷官)은 기본적으로 아들이다.**
    
    **•	여명에서 식신/상관이 없으면 식신/상관이 생한 재성(財星)을 자식으로 본다.**
    
    **③ 남녀 공통 성별 변동 요소**
    
    **자식성의 성별이 변경되는 경우:**
    
    1.	**相穿(상충穿), 空亡(공망), 入墓(입묘), 虛透(허투) 하면 성별이 바뀐다.**
    
    2.	**암장(暗藏)된 자식성은 공망을 적용하지 않으며, 相穿이 있어도 합이 있으면 적용하지 않는다.**
    
    3.	**남명에서 자식성이 자식궁(時支)에 있으면 무조건 아들이 된다.**
    
    4.	**자식성이 천간에 허투(虛透)하면 성별이 변경된다.**
    
    **•	예시: 甲日干이 庚午를 만나면 원래 庚은 아들이지만 허투하므로 딸이 된다.**
    
    5.	**출생 연도에 자식성이 絶地(절지)에 위치하거나 극(剋)을 받으면 성별이 바뀐다.**
    
    **3. 자녀 관계 및 자식운 판단 시 주의점**
    
    **① 사주에서 자녀운을 판단할 때의 기본 원칙**
    
    **•	남명에서 관고(官庫), 여명에서 식상고(食傷庫)가 있으면 자식 운이 좋지 않다.**
    
    **•	남명에서 재성과 관성이 없으면 남자아이를 보기 어렵다.**
    
    **•	남명에서 일지(日支)의 편관은 음관(陰官)이라도 아들로 본다.**
    
    **•	남명의 자식궁(時支)에 묘고(墓庫)가 있으면 배우자성(妻星)과 묘고(墓庫)가 합해야 자식이 생긴다.**
    
    **•	묘고(墓庫): 辰, 戌, 丑, 未 시에 태어난 사람**
    
    **•	단, 자식궁의 묘고가 자식성 자체이거나 배우자성이면 별도의 합 없이도 자식을 가질 수 있다.**
    
    **•	예) 水/木 日干이 時支에 墓庫를 보면, 자식성이 자식궁에 도달해야 자식이 된다.**
    
    **② 자식성(子息星)과 자식궁(子息宮)의 관계**
    
    **•	모든 육친(부모, 배우자, 자식 등)은 반드시 星(성)과 宮(궁)의 관계로 판단.**
    
    **•	자식궁과 자식성이 生(생), 合(합), 伏吟(복음), 入墓(입묘), 祿(록), 原神(원신) 등의 관계가 있어야 한다.**
    
    **•	형(刑), 충(沖), 파(破), 穿(천) 등의 작용도 특수한 경우에 적용될 수 있다.**
    
    **•	자식궁에서 투간(透干)된 오행(五行)이 자식이면, 자식성이 자식궁에 도달한 것으로 본다.**
    
    **•	자식성이 투출되지 않고 지장간(地藏干)에만 있을 경우, 자녀운이 약할 수 있다.**
    
    **4. 자녀 성별을 최종적으로 판단할 때 적용할 원칙**
    
    1.	성별 판단 시 **최초 성별을 기본으로 하되,**
    
    2.	**入墓(입묘), 空亡(공망), 虛透(허투), 穿(천) 등의 조건이 중복될 경우 한 가지만 적용하여 성별을 결정.**
    
    3.	**자식성의 지장간(地藏干)에 투출(透出)된 천간(天干)도 자식으로 본다.**
    
    4.	**子, 午, 卯, 酉는 庫(창고)를 보면 入墓하지 않지만, 남녀 성별 구분 시는 入墓하는 것으로 본다.**
    
    **5. 결론**
    
    **수암명리에서는 사주원국을 통해 자식과의 관계를 자식궁과 자식성을 기준으로 해석한다.**
    
    **•	자식의 성별은 남명에서는 관성, 여명에서는 식신·상관을 기준으로 판단하며,**
    
    **성별은 입묘, 공망, 허투, 상충 등의 영향으로 변경될 수도 있다.**
    
    **•	또한, 자식 운을 볼 때 자식성과 자식궁 간의 관계(생, 합, 극, 입묘 등)를 분석하여 자식과의 인연과 복을 예측한다.**
    
    **즉, 자식궁(子息宮)과 자식성(子息星)이 서로 어떻게 상호작용하는지에 따라 자녀의 유무, 성별, 인연의 강약이 결정된다.**
    

자녀관계

**자녀 중에서 남녀의 성별을 구분하는 기본법**

- 男命에서는 官이 자식이며 자식星이 陽이면 아들, 陰이면 딸이다.
- 男命에서 官이 없으면 財를 자식으로 본다.
※ 男命에 官이 있으면 官이 生한 印星도 자식으로 본다.
- 女命에서는 食/傷官이 자식이며 食神은 딸, 傷官은 아들이다.
(단, 食神이라도 陽 食神이면 아들이다.)
- 女命에서 食傷이 없으면 食傷이 生한 財를 자식으로 본다.
- 男女 구분 없이 地支에 暗藏(암장)된 육친도 자식으로 본다.

**男女의 성별이 바뀌는 경우** 

- 男女 공통으로 자식星이 相穿, 空亡, 入墓, 허투하면 성별이 바뀐다.
(단, 암장된 자식은 空亡을 미적용, 穿에 합이 있으면 穿도 미적용)
- 男命에 자식星이 자식宮에 있으면 陰陽에 구분 없이 아들이다.
- 男命에서 자식星이 財의 生을 받으면 아들, 못 받으면 딸이다.

 예)甲戌日柱가 戌중 辛금은 陰官으로 딸이 되나 財인 戊土의 **生을 받으니** 아들이 된다.

- 男命에서 자식星이 天干에 허투한 경우 성별(性別)이 바뀐다. 예)甲日干이 庚午를 만나면 庚이 아들인데 허투하니 딸이다.
- 원국에 아들이라도 출생년도에 자식星이 絶地(절지)에 임하고, 극(剋)을 받으면 성별이 바뀌어 딸이 된다.

🔘  **四柱(사주)에서 자녀를 판단할 때 주의할 사항**

1. 男命에서 官庫, 女命에서 食傷庫가 있으면 자식 運이 좋지 않다.
2. 男命에서 財/官이 없으면 남자아이로 본다.
3. 男命에서 日支의 편관은 陰官이라도 아들이다.
4. 男命의 자녀궁에 묘고가 있으면 妻星과 墓庫가 합을 해야 자식이 생긴다.
5. ※ 자녀궁에 묘고란?: 辰,戌,丑,未時에 태어난 사람을 의미
6. ※ 男命에 자식宮에 묘고가 있어도 묘고가 妻星이거나 자식星이면 자식宮의 墓庫가 妻星과 합이 없어도 자식을 낳을 수 있다.
예) 水/木 日干이 時支에 墓庫를 보는 경우 자식星이 자식宮에 이르러야 자식이다.

 ✔️모든 육친(부모, 배우자, 자식 등)을 구분할 때는 반드시 星과 宮의 관계를 살핀다.

**星과 宮의 관계란**?
→生,合,伏吟,入墓,祿과 原神 관계 등을 말한다.
→특별한 경우는 刑,沖,破,穿 등도 포함된다.

- 宮과 星이 관계가 없으면 해당 육친이 아니다.
- 자식宮에서 투간(透干)된 五行이 자식이면 자식星이 자식宮에 도달한 것으로 본다.
- 子,午,卯,酉는 庫를 보면 入墓하지 않지만 男女 성별 구분 시는 入墓하는 것으로 본다.
- ▲ 최초 성별(性別) 구분은 현 상태,入墓,空亡,虛透(허투), 穿은 중복되더라도 한 가지만 적용해 성별을 판단한다.
- 자식星의 支藏干(지장간)에서 투간(透干)된 天干도 자식이다

import re

# 1. 텍스트 파일 불러오기
with open('input.txt', 'r', encoding='utf-8') as f:
    text = f.read()

# 2. 단어 치환
text = text.replace("자식궁", "zsg").replace("자식성", "zss")

# 3. 규칙 추출 (예: "zsg이 A일 경우 B가 가능하다.")
pattern = r"zsg[이가]* ([^ ]+)일 경우 ([^\.]+)가 가능하다"
rules = re.findall(pattern, text)

# 결과 출력
print("치환된 텍스트:\n", text)
print("\n[추출된 규칙]")
for a, b in rules:
    print(f"조건: zsg = {a} → 결과: {b}")
